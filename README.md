Purpose:
Developed a React-based web application for an institute, providing an interactive chess interface and a member registration form.
Members can fill out and submit forms directly through the site, demonstrating practical front-end form handling and user interaction.the institute can choose players according to ELO rating and teach them in order.

Technologies:
React.js/HTML/CSS/JavaScript

Tools:
Git & GitHub/VS Code/Browser DevTools

Outcomes:
✍Implemented member registration form allowing users to submit information securely through the site.
✍Added form validation and event handling to improve UX and data accuracy
✍Delivered a project for an institute that enhanced member engagement and streamlined the submission process.
